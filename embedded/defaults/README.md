# Thoughts ({{.Profile}})

A central repository for developer thoughts, notes, research, and context across
projects.

Managed by [thts](https://github.com/scottames/thts).

## Structure

- [`{{.ReposDir}}/`](./{{.ReposDir}}/) - Project-specific thoughts (one subdirectory per project)
- [`{{.GlobalDir}}/`](./{{.GlobalDir}}/) - Cross-project thoughts shared across all repositories

## Usage

```bash
thts init      # Initialize thoughts in a project
thts sync      # Sync changes to this repo
thts status    # Check status
```

See [thts documentation](https://github.com/scottames/thts) for more
information.
